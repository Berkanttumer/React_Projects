# Berkantflix

https://berkantflix.vercel.app/

A Responsive movie and Tv show app used with tmdb api.

### What did i use for this project:

- Vite + React
- React Router
- Tailwind CSS
- Material UI
- Swiper
- Context API
- Firebase
